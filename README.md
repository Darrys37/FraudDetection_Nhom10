# Báo cáo Project Nhóm 10 - Phát hiện Gian lận Giao dịch

## 1. Thành viên nhóm
- Đỗ Xuân Hương (Decision Tree)
- Trần Tất Phát (Random Forest)
- Nguyễn Huỳnh Anh Tuấn (XGBoost)

## 2. Cài đặt môi trường
Yêu cầu Python 3.8 trở lên.
Chạy lệnh sau để cài các thư viện cần thiết:
pip install -r requirements.txt

## 3. Cách chạy chương trình
Để thực hiện toàn bộ quy trình (Xử lý dữ liệu -> Train Model -> Đánh giá):
python src/main.py

## 4. Cấu trúc thư mục
- data/: Chứa file transactions.csv
- src/: Mã nguồn xử lý và huấn luyện
- N10_report.pdf: Báo cáo chi tiết kết quả
